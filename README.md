# Homebrew Tap

[![CI](https://github.com/branchv/homebrew-tap/actions/workflows/ci.yaml/badge.svg)](https://github.com/branchv/homebrew-tap/actions/workflows/ci.yaml)

## Installation

```sh
brew install branchv/tap/<formula>
```

or

```sh
brew tap branchv/tap
brew install <formula>
```
